# SimplePlan

## 开始

SimplePlan是用于记录待办任务的一款挂件，开发的原因主要是想有日历视图。

* 该挂件的任务分为三类：待做任务（todo），正在进行的任务（doing），已经做完的任务（done）
* 每个任务都有自己的一些属性（名称、描述、时间等）
* 通过看板视图和日历视图都能查看任务
* 任务数据存放在块的属性中，所以千万别删除块。后面会增加导出功能。

目的是想做一个待办任务/清单，可以和siyuan深度融合的挂件（插件，看后面支不支持）。

有问题或建议可以提Issues
